# Doc-AI

An AI-powered document summarizer web application that converts lengthy text into concise and readable summaries.

## Features

- Summarize long-form text instantly
- Clean and responsive user interface
- Fast client-side interaction
- Simple document/text input workflow
- Easy-to-use web experience

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Purpose

This project was built to practice frontend development while creating a practical AI-inspired productivity tool for document summarization.

## Learning Outcomes

Through this project, I practiced:

- DOM manipulation
- Event handling
- Frontend UI development
- User input processing
- Application workflow design
- Deploying projects with GitHub Pages

## Future Improvements

- Real AI API integration
- PDF upload support
- Copy summary feature
- Summary length controls
- Dark mode
- Export summarized text
